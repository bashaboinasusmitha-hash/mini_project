# mini_project
📌 Overview
A simple project to analyze sales data using PySpark, with Pandas for processing and Matplotlib for visualization.

⚙️Tech Stack
Python, PySpark, Pandas, Matplotlib, Seaborn

🚀 Features
Created and loaded sample sales dataset
Processed data using PySpark DataFrame operations
Calculated Revenue (Price × Quantity)
Performed product-wise sales analysis
Visualized results using bar and pie charts

📊 Output
Product revenue comparison
Revenue distribution visualization
▶️ Run
pip install pyspark pandas matplotlib seaborn
python your_script.py
